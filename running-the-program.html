Running Your First Ehhlang Program

To run the Ehhlang program, you need to have the Ehhlang compiler installed on your machine. You can download the Ehhlang compiler from the Ehhlang Compiler link. Here are the steps to run your first Ehhlang program:

  1. Download the Ehhlang Compiler: Download the Ehhlang compiler from the link provided above.

  2. Install the Compiler: Install the Ehhlang compiler on your Windows machine.

  3. Create a New Ehhlang Program: Create a new Ehhlang program using a text editor.

  4. Save the Program: Save the Ehhlang program with the .ehh extension.

  5. Run the Program: Open the terminal and run the following command:

    ehhmake <program_name>.ehh

    Replace <program_name> with the name of your Ehhlang program.

  6. View the Output: Once the program is executed, you can view the output image in the same directory.

That's it! You have successfully run your first Ehhlang program. You can now explore the Ehhlang syntax and create more complex programs to generate amazing images.
